オムニローバー3WDサンプルプログラム
=========================
このプロジェクトはワイヤレスロボット制御ボード「V-duino」を搭載した台車型ロボット「オムニローバー3WD」のサンプルプログラムです。

サンプルプログラム一覧
----------------------
本プロジェクトで用意されているサンプルプログラム一覧は下記の通りです。
1. omnirover3wd_auto_motion  
決まった動作を繰り返すサンプル
1. omnirover3wd_HTML_Controller  
スマートフォンやPCから操作するサンプル
